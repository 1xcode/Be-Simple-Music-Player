# Description

Be Simple Music Player is an extremely minimal local Music Player based on
<a href="https://f-droid.org/packages/com.iven.musicplayergo/">Music Player GO</a>! :)

# Features

- Minimalistic interface
- Equalizer
- Light, dark, automatic themes
- Simple audio focus and headsets management
- Search, sorting, shuffle ...
- Modern code base and ... Kotlin!
