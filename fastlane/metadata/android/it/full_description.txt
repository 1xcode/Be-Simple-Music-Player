# Descrizione

Be Simple Music Player è un lettore musicale estremamente minimale derivato da 
<a href="https://f-droid.org/packages/com.iven.musicplayergo/">Music Player GO</a>! :)

# Caratteristiche

- Interfaccia minimalista
- Equalizzatore
- Temi chiaro, scuro, automatico
- Semplice gestione del focus audio e delle cuffie
- Ricerca, ordinamento, shuffle ...
- Code base moderna e ... Kotlin!
